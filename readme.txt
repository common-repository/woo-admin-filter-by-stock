== Description ==

Looking for a tool to filter your WooCommerce products by Stock?

Try Woo Admin Product Stock Filter – it will help you easily filter and list products by Stock in your WooCommerce Admin Dashboard.

The plugin is developed and supported by <a href="https://www.extreme-idea.com/">Extreme Idea LLC</a>. Our entire team is ready to help you. Ask your questions in the support forum, or <a href="https://www.extreme-idea.com/contact-us/">contact us directly</a>.

== Installation ==

Installation

1. Log in as an administrator to WordPress Admin Panel.
2. Navigate to Plugins > press Add New button > press Upload Plugin button.
3. Browse for plugin.zip archive > press Install Now button.
4. The plugin should be successfully installed. Activate the plugin.

How it works

1. Open WP Admin Panel > Products > Select Stock filter operator > Enter quantity > Press Filter.

Available Rule Operators: Stock equals to (=), Stock is less than (<), Stock is greater than (>), Stock is equal to or greater than (≥), Stock is equal to or less than (≤)

For example,  
Select Stock filter operator to '=', quantity='100' press Filter. Results ---> plugin will filter and list all products with product quantity=100.

== Uninstallation ==

To uninstall the plugin:

1. Log in to control panel as administrator.
2. Go to Plugins > Installed plugins > Woo Admin Filter by Stock.
3. Deactivate the plugin.
4. Delete the plugin.

== Upgrade Notice ==

To update the plugin:

1. Log in as administrator to WordPress Admin Panel.
2. Navigate to Plugins > Installed Plugins > Woo Admin Filter by Stock.
3. Press Deactivate button. Press Delete button. The plugin should be successfully uninstalled.
4. Navigate to Plugins > Add New > Upload Plugin.
5. Browse for 'Woo Admin Filter by Stock'.zip file > press Install Now button.

The plugin should be successfully re-installed.
